AI Sentiment Analysis API
A lightweight, scalable Natural Language Processing (NLP) service built with Python and Flask. This API uses the TextBlob library to analyze the emotional tone of text inputs, providing a sentiment verdict (Positive, Negative, or Neutral) and a numerical polarity score.
